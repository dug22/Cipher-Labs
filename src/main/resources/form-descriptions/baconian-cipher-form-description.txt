The Baconian cipher (or Bacon's cipher) is a method of steganographic message encoding, invented by Francis Bacon. Unlike standard encryption, which scrambles a message with a key, this cipher was designed to hide secret messages in plain sight. Bacon's cipher converts every letter in the message to a 5-character binary string or a 5-character letter-encoded string using A and B. Each letter in the alphabet has an assigned 5-character binary string or letter-encoded string. There are two versions of the encoding alphabet. From the earliest model, the letters I and J shared the same encoding, as do the letters U and V. In the second version, the letters I, J, U, and V have their own distinct 5-character encoding.
