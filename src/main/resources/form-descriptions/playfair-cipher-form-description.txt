The Playfair Cipher is a symmetric-key encryption algorithm that processes plaintext into pairs of letters rather than individual characters. The Playfair cipher follows a set of rules. First, you start with a 5x5 Polybius square, writing each key’s character at  the start of the matrix, then finish off with the rest of the alphabet. Do not include the letter J, though it is sometimes combined with the letter I. Next, before encrypting, you must break the plaintext into two-letter pairs. If a pair consists of matching letters, a filler character is then inserted between them. If the final message consists of an odd number of characters, appending a filler character is a must to break even.
