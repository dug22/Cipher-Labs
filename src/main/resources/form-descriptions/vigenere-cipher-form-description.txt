The Vigenère cipher is a poly alphabetic substitution cipher in which each plaintext character is shifted using a repeating key or keyword. Each letter from the key dictates the shift amount for each character in the plaintext. The key cannot be longer than the plaintext, but the key can be shorter than the plaintext. If the key is shorter than the plaintext, the key follows a repeated sequence until it matches the length of the message. With the Vigenère cipher, in terms of positioning, A=0, B=1, …, Z=25. For example, if your message was DOGS, and your key was CAT, D is paired with C (D(3) + C(2)), O with A (O(14) + A(0)), G with T (G(6) + T(19)), and S with C (S(18) + C(2)). Note, if a total exceeds 25, the sequence simply wraps around to the beginning of the alphabet. Now that each plaintext character has been paired with an appropriate character from the key, you would add each pair together to get the encrypted message. The encrypted message would become FOZU. The decryption process follows the same concept, just pair each character from the ciphertext with the appropriate character from your key, then subtract.
