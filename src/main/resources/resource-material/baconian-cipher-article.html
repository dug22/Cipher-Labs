Details Behind Baconian's Cipher

Brief History About the Baconian Cipher

The Baconian cipher, also known as "Bacon's cipher" is a method of steganographic message encoding, designed by Francis Bacon in 1605. Steganography differs from simple encryption. Steganography conceals the presentation of plaintext, rather than its overall content. Think of Bacon's cipher as a concealment cipher.

How Does Bacon's Cipher Work

To encode a plaintext message, each character (letters A-Z) of the plaintext are replaced with a collection of characters that consists the letters A or B. Characters of the plaintext can also be replaced with a 5-bit binary encoding. Down below is a table that consists of letters, and its appropriate letter code.

Encoding A Message Using Bacon's Cipher

Encoding a message using Bacon's cipher is pretty self-explanatory. For this example, we'll encode the message "SECRET" using Bacon's cipher. To encode this message using Bacon's cipher, we must follow these rules:

  1. Break up each character.
  2. Refer to the translation table above to see what each letter's letter code is.
  3. After translating each letter to its appropriate letter code, or binary representation, put it all together

Example down below following the rules above:

  • SECRET
  • S,E,C,R,E,T
    1. S = BAAAB
    2. E = AABAA
    3. C = AAABA
    4. R = BAAAA
    5. E = AABAA
    6. T = BAABA
  • Final Result: BAAABAABAAAAABABAAAAAABAABAABA

Decoding A Message Using Bacon's Cipher

Decoding a message using Bacon's cipher follows the same practice, except you're doing the opposite by translating each encoded chunk back to its appropriate letter.

  1. Break up each encoded chunk by 5.
  2. Refer to the translation table above to see what each letter code's letter is.
  3. After translating each letter code to its appropriate letter, put it all together

Example down below following the rules above:

  • BAAAB AABAA AAABA BAAAA AABAA BAABA
    1. BAAAB = S
    2. AABAA = E
    3. AAABA = C
    4. BAAAA = R
    5. AABAA = E
    6. BAABA = T
  • Final Result: SECRET

Hiding Secret Messages In Sentences

We went over the basic encoding/decoding practices of the Bacon cipher, but we never went over the steganographic concept of it. Francis Bacon was known for concealing messages by using different typefaces embedded within ordinary text. Let's demonstrate this technique using an example. For this example, we'll encode the word "HI" in the sentence "HELLO WORLD". Letters that are bolded resemble the letter "A", while italicized text resembles the letter "B".

  1. Message: HELLOWORLD
  2. Bacon's Encoding: AABBBABAAA
  3. Hidden Word: HI
