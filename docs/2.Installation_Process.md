# <div align="center">Installation Process</div>

Cipher Labs was packaged using jpackage. Since the software was packaged using jpackage, Java does not have to be installed on the main system, because a custom JRE folder comes equipped with the build. Downloading Cipher Labs is a pretty simple process, and here are the steps to follow:
 <ol>
    <li>Navigate to Cipher Labs <a href="https://github.com/dug22/Cipher-Labs/releases">releases tab</a></li>
    <li>Choose the version you wish to use.</li>
    <li>Install the file labeled "CipherLabs.zip" down below.</li>
    <li>Right click the zip file and extract all of its contents.</li>
    <li>Locate the exe file called "CipherLabs.exe" and click on it. Heads up: Do not move the program out of the folder, or else it won't run.</li>
    <li>Enjoy using the program.</li>
</ol>
